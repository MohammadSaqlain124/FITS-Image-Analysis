# 🌌 FITS Image Analysis Pipeline (Orion Nebula - M42)

## 📖 Overview
This project implements a complete scientific pipeline to analyze astronomical FITS images using Python.

It demonstrates how raw telescope data can be processed, visualized, and analyzed to extract meaningful insights.

---

## 🚀 Features

- 📂 FITS file handling using Astropy  
- 📊 Logarithmic contrast enhancement  
- 📈 Pixel intensity distribution analysis  
- 🔍 Bright region detection (feature extraction)  
- 📊 Multi-panel scientific visualization  

---

## 🛰️ Dataset

- Object: Orion Nebula (M42)  
- Format: FITS (Flexible Image Transport System)  
- Source: Digitized Sky Survey  

---

## 🧠 Key Concepts Used

- Image normalization  
- Log scaling  
- Statistical analysis (mean, min, max)  
- Threshold-based segmentation  
- Data visualization  

---

## 📸 Output

The pipeline generates:

- Log-scaled nebula image  
- Pixel intensity histogram  
- Bright region detection overlay
